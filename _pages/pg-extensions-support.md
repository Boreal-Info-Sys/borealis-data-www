The following list of [PostgreSQL extensions](https://www.postgresql.org/docs/17/extend-how.html) are supported by [Borealis Isolated Postgres](https://elements.heroku.com/addons/borealis-pg) add-on databases. Use the [borealis-pg-cli](https://www.npmjs.com/package/borealis-pg-cli) utility to install or remove extensions for an add-on database.
